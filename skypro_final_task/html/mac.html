Терминал MacOS

  • ls - Вывести список файлов и папок в текущей директории
  • cd - Переместиться между папками
  • pwd - Вывести текущую рабочию директорию
  • mv - Удаляет файл или папку из старого местоположения и помещает их в новое местоположение.
  • mkdir - Создать новую папку в текущей директории.
